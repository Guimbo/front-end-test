## Dependências Utilizadas
* React
* Font-Awsome

## Como Usar
 instalar as dependências do projeto
```bash
$ npm install
```

executar projeto no ambiente de desenvolvimento
```bash
 $ npm start
```

compilar para ambiente de produção
```bash
 $ npm run build
```
